Embed HTML Building Block

Allows to embed HTML snippets to your website

Do not limit yourself with default building blocks. With Embed HTML Building Block you can embed new HTML snippets taken from sites like bootsnipp.com.

Website Editor Integration

Embed HTML snippets, remove and edit them directly via editor

A full website editor integration allows you to embed HTML snippets, remove and edit them directly via editor.

Check out this video (https://youtu.be/2NUtSJ9SnJw) showing how to embed a timeline snippet from bootsnipp.com into website.

Another example in this video (https://youtu.be/_KuS8TudfRg) demonstrates how to import another bootsnipp.com snippet and modify it.

Save Templates

Save your HTML snippets as templates and reuse them

Example in this video (https://youtu.be/-8gzwYWX2mo) demonstrates how you can create templates from your HTML snippets and reuse them in other pages without copy&paste.

Installation

Installs on both private and Online/SAAS/Cloud environments

To install this module on private server, you need to:

  • Upload module to your addons folder
  • Restart odoo service
  • Upgrade modules list
  • Install as usual

To install this module on Online/Cloud/SAAS, you need to:

  • Activate the developer mode
  • Install "Base import module" (technical name: base_import_module) module which adds capability to install external modules
  • Click on "Import Module" link in "Apps" menu
  • Select downloaded module archive file (.zip) and click "Import Module" button. Thats it, enjoy!

Check out this video (https://youtu.be/rrScs14ReMg) which demonstrates the process of importing modules on Cloud/SAAS.
